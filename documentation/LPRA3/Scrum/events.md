# Events

## Daily Meetings

- Create a task named **Daily Meeting**.
- Schedule:
  - **1st week:** At least 2 meetings.
  - **2nd week:** At least 3 meetings.
  - **3rd week:** At least 3 meetings. 
  - **4th week:** Meetings every day.

- **During each meeting, the following should be recorded:**
  - **Date, time, duration, and location** of the meeting.
  - **For each team member**:
    - Register completed tasks, upcoming tasks, and any impediments.
    - Write a brief statement on personal performance or team progress.
    - Update task status on the project board to maintain visibility.
